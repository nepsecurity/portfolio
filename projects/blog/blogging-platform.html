Project Overview

This is a simple blogging platform where users can write blog posts and upload images (like screenshots). The platform is built using HTML, CSS, and JavaScript, and it uses local storage to save blog posts temporarily.

Features

  • Add blog posts with titles, content, and images.
  • View all blog posts on the same page.
  • Responsive design for mobile and desktop.

Technologies Used

  • HTML5
  • CSS3
  • JavaScript
  • Local Storage
